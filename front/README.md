# 前端

纯静态页面，由 FastAPI 托管，无需构建工具。

## 文件

| 文件         | 说明                                               |
| ------------ | -------------------------------------------------- |
| `index.html` | 主页面，聊天界面 + 知识库管理面板                  |
| `app.js`     | 交互逻辑，SSE 流式接收、Agent 思考面板、知识库管理 |
| `style.css`  | 全部样式                                           |

## 功能

- **聊天**：多轮对话，SSE 流式输出，支持生成报告 / 闲聊 / 追问三种模式
- **Agent 思考**：折叠面板展示上下文消解 → 查询改写 → 意图识别全流程
- **知识库管理**：上传 xlsx → 自动切片 → 构建向量库 → 提取元数据，全程 SSE 实时日志

## 部署

FastAPI 已配置静态文件挂载，前端文件放在 `front/` 目录下即可：

```python
# rag_chat_v2.py 中已配置
app.mount("/static", StaticFiles(directory="../front"), name="static")
```

访问 `http://localhost:8000` 即可使用。
